## Обзор проекта
Cистема машинного обучения для прогнозирования ежедневных продаж в аптеках Rossmann с автоматизированным ETL-пайплайном и интерактивным дашбордом.

## Технологический стек
- **Оркестрация**: Apache Airflow
- **ML-фреймворк**: LightGBM + Optuna
- **Визуализация**: Dash/Plotly
- **Обработка данных**: Pandas, NumPy
- **Тестирование**: pytest, unittest
- **Окружение**: WSL2

# Установка

## Клонировать репозиторий
```git clone https://github.com/Vexelll/DemandForecasting.git``` -> ```cd demand_forecasting```

## Создать и активировать виртуальное окружение
```python -m venv venv```(**Для Windows**: ```venv\Scripts\activate```)

## Установить зависимости
```pip install -r requirements.txt```

## Настроить WSL окружение (Windows)
```python scripts/setup_wsl.py``` (p.s. нужно установить WSL2, если не установлен)

# Быстрый старт
1. **Инициализировать Airflow**: ```python scripts/run_airflow.py```
2. **Открыть Airflow UI**: ```http://localhost:8080```
3. **Запустить дашборд**: ```python src/visualization/dashboard.py``` -> ```http://localhost:8050```

# Ключевые возможности
- Автоматизированный ETL-пайплайн на Airflow
- Модель прогнозирования LightGBM (MAPE 9.98% на данный момент)
- Интерактивный дашборд Dash
- Комплексный мониторинг и алертинг
- Юнит- и интеграционные тесты

# Производительность модели
| Метрика | Значение | Описание |
| --------------- | --------------- | --------------- |
| **MAPE** | 9.98% | Средняя абсолютная процентная ошибка |
| **R² Score** | 0.92 | Качество модели |
| **MAE** | 677 € | Средняя абсолютная ошибка |
| **Время инференса**	 | --- | Прогноз для всех магазинов |

# Расписание пайплайна
- **Ежедневный прогноз**: 2:00 (demand_forecasting_pipeline)
- **Еженедельное переобучение**: Воскресенье 3:00 (weekly_model_retraining)

# Возможности дашборда (на данный момент)
- Сравнение прогноза и факта
- Анализ распределения ошибок
- Сравнение магазинов
- Визуализация важности признаков
- Обновление метрик в реальном времени

## Исходные данные
- Данные о продажах: 1115 магазинов
- Метаданные магазинов: локация, тип, ассортимент
- Промо-данные: акции Promo, Promo2
- Календарные события: праздники, школьные каникулы

## Сгенерированные признаки
- **Временные**: DayOfWeek_sin, Month_cos, IsWeekend (15 признаков)
- **Лаговые**: SalesLag_7, RollingMean_28 (12 признаков)
- **Промо**: PromoSequence, DaysSinceLastPromo (8 признаков)
- **Праздничные**: IsPublicHoliday, WasHolidayYesterday (6 признаков)
